# Overview
HireMe is a job search engine utilizing Google's advanced search operators that integrates data from Glassdoor and Payscale.

*Link*: TBD

## Getting Started
Client Side:
```bash
cd client-side
npm install
bower install
grunt compass
grunt uglify
grunt
```

Server Side:
```bash
cd server-side
npm install
node server.js
```

## Built With
Node.js, Express, AngularJS, Foundation, jQuery, slick, [MongoDB, Glassdoor API]
